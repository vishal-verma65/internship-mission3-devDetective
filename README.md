# 🔍 Dev-Detective

A noir-themed GitHub profile explorer built as an **internship task project**.
Investigate any developer's GitHub presence or pit two developers head-to-head in a battle of stats.

live link: https://yourdevdetective.netlify.app/

## ✨ Features

- 🕵️ **Investigate Mode** — search any GitHub username and view their profile, bio, repos, stars, followers and portfolio link
- ⚔️ **Battle Mode** — compare two developers by followers or total stars, with a glowing winner card and verdict
- 🌙 **Dark / Light Theme** — toggle with preference saved to localStorage
- 🕐 **Search History** — recent searches surface in a dropdown on input focus
- 💾 **Session Cache** — last viewed profile is restored on page reload
- 📱 **Fully Responsive** — works cleanly on mobile and desktop

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure and semantic markup |
| CSS3 | Theming, animations, responsive layout |
| JavaScript (ES Modules) | Logic, API calls, DOM rendering |
| Vite | Development server and production bundling |
| GitHub REST API | Live developer data (no auth required) |

---

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
